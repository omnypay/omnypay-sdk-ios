OmnyPayError

public enum OmnyPayError: ErrorType

OmnyPayError is an error type returned by OmnyPay, that tells about specific error that can occurs while accessing OmnyPay APIs. It encompasses a few different types of errors, each with their own associated reasons.

  • requiredDataNotAvailable: Returned when any requried data is not available.
  • dataValidationFailed: Returned when a any data validate fails.
  • RequiredDataNotAvailableReason

    The underlying reason when the requiredDataNotAvailable error occurred.

    • merchantIdNotAvailable: MerchantId is not provided.
    • sessionNotAvailable: A transaction session is not available.
    • authenticatedSessionNotAvailable: An authenticated session is not available.

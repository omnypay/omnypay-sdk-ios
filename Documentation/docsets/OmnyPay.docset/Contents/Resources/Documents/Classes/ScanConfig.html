ScanConfig

public class ScanConfig: NSObject

An instance that represents the configuration of scanner. OmnyPayScan instance can be configured with custom scan mode, code types to scan, whether to start scanning as soon as scanner is presented and a custom dictionary for any other information.

    Parameters

    scanMode

    ScanMode of the scanner. Defaults to Single
    startScanningImmediately

    Pass true to start scanning as soon as scanner is presented; otherwise, pass false. Default value is true
    codeTypes

    Machine code types that a scanner can scan. For valid values refer to AVMetadataObjectType strings defined in AVMetadataObject.h. Default is all AVMetadataObjectType strings.
    userInfo

    A dictionary to pass any custom information. Defaults value is nil.

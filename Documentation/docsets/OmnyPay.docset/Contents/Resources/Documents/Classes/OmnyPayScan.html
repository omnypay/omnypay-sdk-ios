OmnyPayScan

public final class OmnyPayScan: NSObject

OmnyPayScan SDK provides an easy way to scan machine readable codes like QRCode, Barcode etc.

    Parameters

    A view controller over which scan view to be presented
    animated

    Pass true to animate the presentation; otherwise, pass false.
    handler

    The block to execute after the presentation finishes. This block has no return value and takes two parameters. You may specify nil for this parameter. This completion handler takes the following parameters:

    success - A Boolean to indicate whether presentation is successful.

    error - An NSError object that indicates why the presentation failed, or nil if the presentation is successful.
  • dismissScanView(animated:completion:)

    Dismisses the scan view that was presented modally over the view controller passed in presentScanView(over:animated:hadler)

    If scanMode property of receiver object’s config (ScanConfig) is Single then scan view is automatically dismissed once a single code is scanned. In all other cases the presenting view controller is responsible for dismissing the scan view it presented.

    Declaration

    Swift

    Parameters

    animated

    Pass true to animate the dismissal; otherwise, pass false.
    completion

    The block to execute after the scan view is dismissed. This block has no return value and takes no parameter. You may specify nil for this parameter.

    Start the scanning operation.

  • stop()

    Stop the scanning operation.

    If scanMode property of receiver object’s config (ScanConfig) is set to Single then scanning automatically stops as soon as a single code is scanned and scan view is dismissed.

    In case of Batch scan mode call this function to stop scanning explicitly.

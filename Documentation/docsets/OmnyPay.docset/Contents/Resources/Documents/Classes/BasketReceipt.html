BasketReceipt

@objc public class BasketReceipt: NSObject

The BasketReceipt represents a Payment receipt of active shopping session, which includes product line items, discount offers, selected payment instrument, final reconciled totals for selected payment instrument and loyalty redemption status.

  • Tags

    An enum representing tags that can be used for various checkout flows.

    • collect: Represents support for reserving and collecting coupons.

  • State

    An enum representing the state of current basket.

    • CompleteTransaction: A state of basket when transaction is successfully completed.
    • InitiatePayment: A state of basket when app starts the payment.
    • New: A state of a new or already existing active basket.
    • CompleteScan: A state for basket when Point of Sale terminal finishes adding line items.
    • ReadyForPayment: A state of basket between PoS starts payment process and payment vault responds.
    • CancelTransaction: A state of basket when transaction is explicitly cancelled.
    • UpdateBasket: A state of basket when the basket is updated.
    • AssociatePos: A state of basket when transaction will be associated with pos.

OmnyPayScan

OmnyPayScan SDK provides an easy way to scan machine readable codes like QRCode, Barcode etc.

  • OmnyPayScan

    A final class that can be used to scan machine readable codes like QRCode, BarCode etc.

    This class provides a singleton instance that is initialized with a default configuration of ScanConfig type. For most of the purposes this singleton instance should be enough.

    For any custom usage a new instance of this class can be created by providing a custom configuration.

    An instance that represents the configuration of scanner. OmnyPayScan instance can be configured with custom scan mode, code types to scan, whether to start scanning as soon as scanner is presented and a custom dictionary for any other information.

  • ScanMode

    A type that represents the scanning mode of OmnyPayScan

    • Single: Single scan mode. Use this scan mode when a single machine code is to be scanned. Scanner view is dismissed after the scanning completes.
    • Batch: Batch scan mode. Use this scan mode where multiple machine codes are to be scanned. Scanner view continue to scan any code that is detected until stop() is called on OmnyPayScan instance. For each code that is scanned, didScanHandler is executed. Scanner view has to be explicitly closed/removed by calling dismissScanView(animated:, completion:) or removeScanView() on OmnyPayScan instance.

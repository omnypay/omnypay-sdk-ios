OmnyPayPIScan

public final class OmnyPayPIScan: NSObject

A final class that can be used to scan a credit card.

This class provides a singleton instance that is initialized with a default configuration of PIScanConfig type. For most of the purposes this singleton instance should be enough.

For any custom usage a new instance of this class can be created by providing a custom configuration.

    The block to execute after the presentation finishes. This block has no return value and takes two parameters. You may specify nil for this parameter. This completion handler takes the following parameters:

    success - A Boolean to indicate whether presentation is successful.

    error - An NSError object that indicates why the presentation failed, or nil if the presentation is successful.

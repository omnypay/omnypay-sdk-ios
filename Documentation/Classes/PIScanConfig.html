PIScanConfig

public class PIScanConfig: NSObject

An instance that represents the configuration of identity scanner. Apart from documentType and country, a custom dictionary can also be provided for any other information that an application wants to pass.

              cardNumberMaskingEnabled: Bool = true,
              userInfo:[String:AnyObject?]? = nil)

    Parameters

    cvvRequired

    Pass true if CVV is required from user, else pass false.
    expiryDateEditable

    Pass true if expiry date scanned from card should be editable, else pass false.
    cardHolderNameEditable

    Pass true if card holder name scanned from card should be editable, else pass false.
    cardNumberMaskingEnabled

    Pass false if complete card number scanned from card should be displayed while editing card details, pass true if masked card number should be displayed.
    userInfo

    A dictionary to pass any custom information. Defaults value is nil.

    An instance of PIScanConfig
